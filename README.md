# Setup Guid:

### Python setup
To run the python version you are going need to follow those following steps:
** For this setup was used python 3.9.7 **
1. cd python
2. pip install -r requirements.txt
3. python3(or just python) automatos_celulares2.py

### VueJS setup
1. cd vuejs
2. npm install
3. npm run serve
